4ARIA ReAct Agent Architecture

ReAct Loop Flow

User Message
→
Load Short-term Memory
→
mock_llm.create()

tool_use?
├─ YES →
dispatch_tool(name, input)
→
append tool_result
→

└─ NO →
extract final text
→
save_short_term()
→

Tool 1: query_metrics

Parameterized timeseries query with GROUP BY aggregation.
G2 Rejects ranges > 48h

Tool 2: compare_kpi_windows

Delta between Window A and Window B. Returns Degraded/Improved/Stable.
Handles +inf% for zero-baseline codes

Tool 3: detect_anomalies

Z-score vs 24h rolling baseline. Writes to agent_memory.
Warning z≥2.0, Critical z≥3.0

Tool 4: get_top_movers

Top N KPIs by |pct_change|. Delegates to compare_kpi_windows.
Filter: degraded / improved / both

Tool 5: get_current_snapshot

Latest 5-min window only (not a timeseries).
Fastest tool — single-timestamp query

Tool 6: resolve_time_reference

NLP → ISO 8601 (5-min snapped).
Handles: 3PM, 14:00, 2h ago, this morning, yesterday

Tool 7: summarize_panel

Trend (first-half vs second-half), top codes, threshold breaches, narrative.

ScopePer session_id

Written bydetect_anomalies tool

9One-Line Real Claude API Swap

The entire application runs in mock mode by default — no API key needed for the demo. To switch to real Claude, only two values in config.py need changing:
# config.py — change these two lines only
USE_MOCK_LLM = False
ANTHROPIC_API_KEY = "sk-ant-..."
The ReAct loop, tool dispatch, memory, observability, and all panels work identically. The mock_llm.create() function routes to the real SDK automatically.
